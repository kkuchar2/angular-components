Generic Table

Sortable, paginated table with custom cells, height modes, and column toggles. Use the theme toggle in the top bar to preview dark mode.

1. Full featured

Sorting, pagination, column toggle chips, CSV export, built-in uuid/date cells with optional copy buttons, a custom status badge cell, and clickable rows.

2. Server-side pagination

87 rows on the “server”, five per page. Only the current page is passed to [data]; each paginator click triggers a simulated fetch (~350 ms). Export CSV uses (exportRequest) so the parent can load all rows first.

3. Custom cell templates

Multiple projected templates: an initials avatar, a mailto link, and a status badge.

{{ initials(row) }} {{ row.name }} {{ row.email }} {{ row.status }}

4. Virtual scroll

10 000 rows with only the visible slice in the DOM. Sorting still runs client-side over the full dataset. Requires a fixed scroll height and uniform row height.

5. Empty state

Non-virtualized table — a centered message replaces the rows when there is no data.
